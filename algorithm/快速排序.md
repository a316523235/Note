## 注意点
快速排序不是一种稳定的排序算法，也就是说，多个相同的值的相对位置也许会在算法结束时产生变动

## 实现代码
``` php
function quickSort(&$arr) {
    qSort($arr, 0, count($arr) - 1);
}

function qSort(&$arr, $i , $j) {
    $start = $i;
    $end = $j;

    if($i >= $j) {
        return;
    }

    $p = $arr[$i];

    while ($j > $i) {
        while ($j > $i && $arr[$j] >= $p) {
            $j--;
        }
        swap($arr, $i, $j);
        while($j > $i && $arr[$i] <= $p) {
            $i++;
        }
        swap($arr, $i, $j);
    }

    qSort($arr, $start, $i-1);
    qSort($arr, $i+1, $end);
}

function swap(&$arr, $i, $j) {
    $temp = $arr[$i];
    $arr[$i] = $arr[$j];
    $arr[$j] = $temp;
}


$test = [5, 13, 15, 11, 6, 3, 7, 9, 2, 12];
quickSort($test);
print_r(implode(',', $test));
```
